# Configuration Files

## `pinyin.conf`
  Configures the Pinyin input method (Chinese). Settings include:
  - Shuangpin profile and display mode
  - Page size, spell/symbol/chaizi features
  - Cloud pinyin settings
  - Preedit display options
  - Prediction settings
  - Keyboard shortcuts for candidate selection, page navigation
  - Fuzzy matching rules for common typos
  - Quick phrase triggers

## `hangul.conf`
  Configures the Hangul input method (Korean). Settings include:
  - Keyboard layout (Dubeolsik)
  - Auto reorder and stroke combination options
  - Word commit behavior
  - Hanja mode toggle
  - Keyboard shortcuts for navigation

## `chttrans.conf`
  Configures Chinese Traditional/Simplified translation using OpenCC. Settings include:
  - Input method selection
  - OpenCC configuration (s2t.json - Simplified to Traditional)
  - Hotkey (Ctrl+Shift+F) to toggle translation

## `punctuation.conf`
  Controls punctuation handling. Settings include:
  - Enable/disable status
  - Half-width punctuation preferences
  - Custom mapping options
  - Hotkey (Ctrl+period) to toggle

## `classicui.conf`
  Configures the Classic UI appearance and behavior:
  - Candidate list orientation (vertical/horizontal)
  - DPI and mouse wheel settings
  - Fonts for different UI elements
  - Tray icon appearance
  - Theme selection (light/dark)
  - Custom color options for background, text, and highlights

## `clipboard.conf`
  Configures clipboard history functionality:
  - Trigger key settings
  - Primary selection paste option
  - Number of clipboard entries to store (5)
  - Navigation shortcuts

## `notifications.conf`
  Controls which notifications are hidden. Currently no notifications are hidden.

## `cached_layouts` (generated automatically)
  Cache file storing keyboard layout information (very large file, likely autogenerated).
